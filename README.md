# graphql_livestock_auction
Graphql learn and practicing

## Requirements
1. IDE: VS Code.
2. NodeJs Latest.
3. MySQL Workbench Community Edition.
4. Docker.
5. Git.

## Installation
Open the terminal and run the following:

1. Run `npm install yarn -g`.
2. Run `yarn`
3. Run `F5` for debugging. (VS Code *ONLY*)
4. Run `yarn start`.
